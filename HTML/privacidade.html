Política de Privacidade

Última atualização: julho de 2026

1. Dados que recolhemos

Recolhemos apenas informações necessárias para agendamento e contacto, como nome, telefone e email.

2. Como utilizamos os dados

Os dados são utilizados apenas para comunicação com clientes — confirmação de marcações, lembretes e respostas a pedidos de contacto.

3. Partilha de dados

Não vendemos nem partilhamos os teus dados com terceiros para fins de marketing.

4. Os teus direitos

O utilizador pode solicitar a qualquer momento a alteração ou remoção dos seus dados pessoais, entrando em contacto connosco.

Tens dúvidas sobre os teus dados? Fala connosco.
